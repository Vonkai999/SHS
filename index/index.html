Отзывы

Правила

1. Абонемент действует ровно 1 месяц. Абонемент на мини-группы включает 4 занятия, абонемент на курс — 8 занятий

2. Отработка пропущенного занятия возможна только при предупреждении за 24 часа и при наличии свободных мест в группе, уточнять в директ. (действует только для мини-групп). Без предупреждения занятие сгорает

3. По одному абонементу может ходить только 1 человек

4. После покупки абонемента средства не подлежат возврату (исключение – отмена занятия мной)

5. Посещение занятий возможны только по предварительной записи и оплате

6. Индивидуальные занятия проводятся только по предоплате 50%

7. Все хореографии авторские. При публикации в соцсетях, пожалуйста, отмечайте меня.
